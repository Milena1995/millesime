# Design & moodboard — Millésime

## Ton visuel

Chic, bohème, neutre, **old money** — sobre et intemporel plutôt que flashy. On évite les couleurs vives, les gradients criards, les icônes ludiques. On privilégie les textures matières (papier, lin), les lignes fines, beaucoup d'espace blanc/crème.

## Typographie

- **Titres** : serif élégant type *Playfair Display* — évoque les étiquettes de vin haut de gamme.
- **Corps de texte / UI** : sans-serif discret type *Inter* — lisible, neutre, ne concurrence pas les titres.

## Éléments d'UI

- Boutons : coins légèrement arrondis (4–6px, pas de style "app ludique"), fond bordeaux ou contour fin doré.
- Cartes bouteille : ombre très légère, fond blanc cassé, ratio portrait pour mettre l'image en valeur.
- Étoiles de notation : contour fin doré/laiton plutôt que jaune vif.
- Icônes : traits fins (style "line icons"), pas d'icônes remplies colorées.

## Références d'ambiance

- Étiquettes de domaines viticoles français classiques (typographie gravée, cadres fins).
- Intérieurs "old money" : bibliothèques, boiseries, lin naturel, laiton vieilli.
- Papeterie haut de gamme (cartons d'invitation, menus de restaurant gastronomique).

## Application à l'image générée par IA

Le prompt de régénération d'image (voir tech-stack.md) doit demander un fond neutre cohérent avec cette palette (ivoire/beige), une lumière douce et chaude, un cadrage centré identique pour toutes les bouteilles — afin que la galerie ait l'air d'une collection soignée plutôt que d'un ensemble de photos disparates.
